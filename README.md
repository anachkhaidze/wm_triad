# WM Triad

This repository contains all code and materials required to reproduce the study and analysis presented in the [preprint](link once we have it).

### Contents

1. **Analysis scrtipts**
   - Preprocess imagery questionnaire responses: `analysis/imagery_questionnaire_preprocessing.ipynb`
   - Preprocess WM task data and merge it with questionnaires: 'wm_task_qa_merging.ipynb'
   - Preprocess incorrect trials: `analysis/error_analysis.ipynb`
   - Run regression models: `analysis/regression_modeling.ipynb`
   - Demographic analysis: 'demographics_analysis.ipynb'
   - Create plots: `analysis/plots.ipynb`
