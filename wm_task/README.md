# WM Deepgen Experiment

A PsychoPy/Pavlovia experiment for studying working memory with deep generative models.

## Structure

- `WM_Deepgen.psyexp` - PsychoPy experiment file
- `WM_Deepgen.js` - JavaScript version for Pavlovia
- `resources/` - Condition files and task phase definitions
- `stimuli/` - Stimulus images and cues
- `new_stimuli/` - Additional stimulus set
- `data/` - Output directory for participant data

## Running the Experiment

1. Open `WM_Deepgen.psyexp` in PsychoPy Builder
2. For online use, sync to Pavlovia

## Condition Files

- `initial_practice.csv` - Practice trials
- `demo.csv` - Demo trials
- `prac.csv` - Practice phase
- `selected_and_foils.csv` / `selected_and_foils_new.csv` - LTM recognition test
